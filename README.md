## thymeleaf
* 서버 사이드 랜더링(SSR)
* 네츄럴 템플릿
* 스프링 통합 지원

### 서버 사이드 랜더링(SSR)
타임리프는 백엔드 서버에서 HTML을 동적으로 렌더링 하는 용도로 사용된다.

### 네츄럴 템플릿
타임리프는 순수 HTML을 최대한 유지하는 특성

순수 HTML을 그대로 유지하면서 뷰 템플릿도 사용할 수 있는 타임리프의 특징을 
**네츄럴 템플릿(natural templates)** 이라 한다.

### 스프링 통합 지원
타임리프는 스프링과 자연스럽게 통합되고, 스프링의 다양한 기능을 편리하게 사용할 수 있게 지원한다.
